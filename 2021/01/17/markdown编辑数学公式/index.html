markdown编辑数学公式

转义

  • 需要表示标记标记字符时，需要转义\
  • 需要转义的基础字符:# $ % & ~ _ ^ \ { }

分数

  • \frac{}{}
  • 第一个{}分子，第二个分母
  • \dfac{}{}字号为独立公式大小
  • \tfac{}{}字号为行间公式大小
名称 数学表达式 markdown公式
分数 $\frac{a+1}{b+1}$ $\frac{a+1}{b+1}$
分数字号 $\dfrac{1}{2} \tfrac{1}{2}\frac{1}{2}$ $\dfrac{1}{2} \tfrac{1}{2}\frac{1}{2}$

上下标

  • 上标^
  • 下标_
  • 上下标如果多于一个字符，需要用{}括起来
  • 上下标前后顺序可互换
名称 数学表达式 markdown公式
上标 $x^n$ $x^n$
下标 $x_n$ $x_n$
简单混合 $x^2_n$,$x_n^2$ $x^2_n$,$x_n^2$
复合标记 $x_{n+1}^{3n}$ $x_{n+1}^{3n}$

累加

  • \sum_{}^{}

  • 上下标前后顺序可互换

名称 数学表达式 markdown公式
求和 $\sum{x^n}$ $\sum{x^n}$
带范围求和 $\sum_{n=1}^N{x^n}$ $\sum_{n=1}^N{x^n}$

累乘

  • \prod_{}^{}
  • 上下边前后顺序可互换
名称 数学表达式 markdown公式
求积 $\prod{x^n}$ $\prod{x^n}$
带范围求和 $\prod_{n=1}^N{x^n}$ $\prod_{n=1}^N{x^n}$

开方

  • \sqrt[]{}
  • [开方数]，{底数}
  • $\sqrt{x}\quad\sqrt[3]{x}$
    • \quad表示添加空格
    • $\sqrt{x}\quad\sqrt[3]{x}$
名称 数学表达式 markdown公式
开方号 $\sqrt[10]{1024}$ $\sqrt[10]{1024}$

积分

  • \int_{}^{}
  • 若用i数目对应积分重数，则至多4重积分，如前四行
  • 反复使用\int_{}^{}则不受积分重数限制
  • $\idotsint$表示$\idotsint$
名称 数学表达式 markdown公式
积分 $\int_{积分下限}^{积分上限}{f(x)}dx$ $\int_{积分上限}^{积分下限}{f(x)}dx$
二重积分 $\iint_{积分下限}^{积分上限}{f(x,y)}dv$ $\iint_{积分下限}^{积分上限}{f(x,y)}dv$
三重积分 $\iiint_{积分下限}^{积分上限}{f(x,y,z)}dv$ $\iiint_{积分下限}^{积分上限}{f(x,y,z)}dv$
四重积分 $\iiiint_{1}^{2}{f(x,y,z,t)}dxdydzdt$ $\iiiint_{1}^{2}{f(x,y,z,t)}dxdydzdt$
二重积分 $\int_{积分下限}^{积分上限}\int_{积分下限}^{积分上限}{f(x,y)}dxdy$ $\int_{积分下限}^{积分上限}\int_{积分下限}^{积分上限}{f(x,y)}dxdy$
三重积分 $\int_{积分下限}^{积分上限}\int_{积分下限}^{积分上限}\int_{积分下限}^{积分上限}{f(x,y,z)}dxdydz$ $\int_{积分下限}^{积分上限}\int_{积分下限}^{积分上限}\int_{积分下限}^{积分上限}{f(x,y,z)}dxdydz$

斜括号

  • 如果斜除号两边的字符比较高，用常规的 / 会导致式子很不协调
  • 使用 \middle/ 来使得斜除号的高度与两侧字符高度相匹配。
  • \left. +[公式]+\right.
  • $x = \left. a^\frac{1}{2} \middle / b^{\frac{lnx}{x}} \right.$
  • $x = \left. a^\frac{1}{2} \middle / b^{\frac{lnx}{x}} \right.$
